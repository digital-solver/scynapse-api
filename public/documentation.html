My API - Movie List

About Movie List API

This API will allow a front end to get movies to display to the user.

Go back to the homepage

Endpoints

Name Description URL Method Request Body Format Response Body Format
Get All Movies Returns a list of all movies to the user. /movies GET None A JSON object holding data about all the moviies.
Get Single Movie Returns data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user. /movies/:title GET None a JSON object holding data abot a single movie, contining a title, description, genre, actors, composer, ost, and director. Formatted as follows:

{
title: 'Blade Runner',
description: 'A movie about androids in a futuristic world.',
genre: 'Science Fiction',
actors: {
  1: 'Harrison Ford',
  2: 'Rutger Hauer',
},
composer: 'Vangelis',
ost: {
  1: 'Main Titles',
  2: 'Blush Response',
  3: 'Wait for Me'
},
director: 'Ridley Scott',
}
Get Genre Return data about a genre (description) by name/title (e.g., “Thriller”). movies/:genre GET None A text message containing a description about a genre.
Get Director Return data about a director (bio, birth year, death year) by name. /movies/:director GET None A JSON object containing data about a director. Formatted as follows:

{
name: 'John Smith',
bio: 'A famous director, known for good movies.',
dateOfBirth: '11-04-1956',
dateOfDeath: '-',
}
Register User Allow new users to register. /users POST A JSON Object Holding data about the user to add. Structured as follows:

{
name: 'John Smith',
username: 'jsmith99',
password: '1234',
} 		A JSON Object Holding data about the user added. Structured as follows:

{
name: 'John Smith',
username: 'jsmith99',
password: '1234',
id: 'e13d345-1d467e89-1d246e8-12d89e9',
}
Update User Allow users to update their user info (username). /users/:username PUT None A text message indicating whether the info was successfully updated.
Deregister User Allow existing users to deregister (showing only a text that a user email has been removed—more on this later). /users/:username DELETE None A text message indicating whether the user was successfully deregistered.
Add Movie to Favourites Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later). /users/:username/favourites/:title POST None A text message indicating whether the movie was successfully added to favourites.
Remove Movie from Favourites Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later). /users/:username/favourites/:title DELETE None A text message indicating whether the movie was successfully removed from favourites.
