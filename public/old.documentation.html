My API - Movie List

About Movie List API

This API will allow a front end to get movies to display to the user.

Go back to the homepage

Endpoints

Name Description URL Method Request Body Format Response Body Format
Get All Movies Returns a list of all movies to the user. /movies GET None A JSON object holding data about all the movies.
Get Single Movie by Title Returns data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user. /movies/:Title GET None a JSON object holding data about a single movie, contining a title, description, genre, actors, composer, ost, and director. Formatted as follows:

{
title: 'Blade Runner',
description: 'A movie about androids in a futuristic world.',
genre: 'Science Fiction',
actors: {
  1: 'Harrison Ford',
  2: 'Rutger Hauer',
},
composer: 'Vangelis',
ost: {
  1: 'Main Titles',
  2: 'Blush Response',
  3: 'Wait for Me'
},
director: 'Ridley Scott',
}
Get Genre Return data about a genre (description) by name/title (e.g., “Thriller”). genres/:Genre GET None A JSON object containing the name of and description about a genre.
Get Director Return data about a director (bio, birth year, death year) by name. /directors/:Director GET None A JSON object containing data about a director. Formatted as follows:

{
Name: 'John Smith',
Bio: 'A famous director, known for good movies.',
Birth: '11-04-1956',
Death: '-',
}
Create User Allow new users to register. /users POST A JSON Object Holding data about the user to add. Structured as follows:

{
Username: 'jsmith99',
Password: '1234',
Email: 'example@example.com',
} 		A JSON Object Holding data about the user added. Structured as follows:

{
Username: 'jsmith99',
Password: '1234',
Email: 'js@example.io',
FavoriteMovies: [],
_id: '62ec1f3b8725f1f24d724804',
__v: '0',
}
Update User Allow users to update their user info (username). /users/:Username PUT A JSON object containing the updates to be made. Formatted as follows:

{
Username: 'jsmith99',
Password: 'UPDATED_PASWORD',
Email: 'js@example.io'
} 		A JSON object of the user containing the new data.
Remove User Allow existing users to deregister (showing only a text that a user email has been removed). /users/:Username DELETE None A text message indicating whether the user was successfully deregistered.
Add Movie to Favourites Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later). /users/:Username/favorites/:MovieID POST None A JSON object showing the MovieID data in the users favorite movies array.
Remove Movie from Favourites Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later). /users/:Username/favorites/:MovieID DELETE None A JSON object showing that the MovieID is not in the users favorite movies array.
